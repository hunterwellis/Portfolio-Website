6-DOF Robot Arm
Robotic arm designed to train RL algorithms. Using a ROS2 and Gazebo simulation environment with stepper motors, pulleys, and embedded controls.
Project Details / Background
Hardware:
The structural components of this build are all 3D printed with PLA. The motors are stepper motors taken from an old 3D printer. To control the motors, I used the same old 3D printer's stepper motor driver and wrote custom firmware to interact with the stepper motors and magnetic encoders on the arm. Eventually, the motor controllers I am designing as part of another project will control the arm.

Stepper Motor Controller
Software:
This project uses ROS2 and Gazebo to create a simulation environment and communicate over serial with the hardware. There are 6 ROS2 packages currently,

arm_description -- Contains robot URDF, meshes, and launch files for RViz and the robot state publisher.

arm_simulation -- The simulation package contains the SDF world files and other files related to setup of the Gazebo simulation environment.

arm_rl -- The reinforcement learning package has launch files to set up gym training environments and scripts to run RL algorithms.

arm_cv -- Computer vision package using YOLOv8 (with oriented bounding boxes).

arm_nlp -- Natural language processing package for converting natural language prompts to symbolic representations.

arm_link -- Serial communication link to arm.
